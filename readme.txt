<p>#FrontendTT2024</p>
<p>## Descripción</p>
<p>Este proyecto es una página web empresarial diseñada con HTML5 y CSS3, enfocada en ofrecer una experiencia de usuario responsiva y estética. La página incluye secciones de Productos, Reseñas y Contacto, con un formulario funcional para que los visitantes puedan comunicarse.</p>
<p>## Características</p>
<ul>
<li>**Estructura Semántica**: Utiliza etiquetas semánticas como <code>&lt;header&gt;</code>, <code>&lt;nav&gt;</code>, <code>&lt;main&gt;</code>, <code>&lt;section&gt;</code> y <code>&lt;footer&gt;</code> para mejorar la accesibilidad y el SEO.</li>
<li>**Formulario de Contacto**: Implementado con Formspree para manejar el envío de datos de manera sencilla.</li>
<li>**Diseño Responsivo**: Adaptable a diferentes tamaños de pantalla mediante Flexbox, Grid y Media Queries.</li>
<li>**Estilos Personalizados**: Utiliza Google Fonts y propiedades de background para un diseño atractivo.</li>
<li>**Multimedia Integrada**: Incluye imágenes y videos embebidos para enriquecer el contenido.</li>
</ul>
